# Ponderr TV - Digital Learning Portal
**A Range Informatics Initiative**

## Project Overview
Ponderr TV is an interactive PWA designed to deliver K-12 and Professional education. It leverages YouTube for video hosting and GitHub for high-speed, low-cost deployment.

## Architecture
- **Root**: Contains PWA essentials (`manifest.json`, `sw.js`).
- **assets/**: All branding and UI elements.
- **members-access/**: Gated content and downloadable resources (PDFs/Printables).
- **js/**: Platform detection and interactive UI logic.

## Deployment
1. Upload to the `RangeInfo-TV` GitHub Organization.
2. Connect `ponderr.tv` via GitHub Pages settings.
3. Manage subscribers via Razorpay redirection.
